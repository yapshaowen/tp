# EstateContacts

[![CI Status](https://github.com/AY2526S2-CS2103-F13-3/tp/workflows/Java%20CI/badge.svg)](https://github.com/AY2526S2-CS2103-F13-3/tp/actions)

![Ui](docs/images/Ui.png)

**EstateContacts** is a desktop application designed for real estate agents to manage their client and property contacts efficiently.

* It is optimised for use via a **Command Line Interface (CLI)** while still having a **Graphical User Interface (GUI)**.
* It helps real estate agents keep track of clients, properties, and related contact details in one place.
* Built in Java with an OOP architecture, it provides a well-structured codebase (~6 KLoC) with user and developer documentation.

For detailed documentation, visit the **[EstateContacts Product Website](https://ay2526s2-cs2103-f13-3.github.io/tp/)**.
